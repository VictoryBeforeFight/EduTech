HTML 폼 구조

form

<form>은 block 요소이며, <form>을 제외한 <label>, <input>, <textarea>, <button> 등은 inline 요소 입니다.

Form Attributes

action
입력 결과를 보낼 주소를 표시합니다.
method
GET, POST, PUT, PATCH, DELETE 요청이 가능합니다.
enctype
데이터 타입을 명시합니다.

예시

폼 구성요소

lable

앞의 텍스트를 클릭해도 동작한다.

<input> 태그와 연결되어 해당 인풋에 대한 설명을 제공합니다. <label> 태그의 for 속성값과, <input> 태그의 id를 일치시켜 활용합니다.

예시

label 태그 장점

1. 스크린 리더 사용자들에게 해당 인풋이 어떤 인풋인지 알려주는 역할을 합니다.

2. 사용자가 레이블을 클릭하는 경우 웹 브라우저는 자동적으로 인풋 태그에 포커싱을 해줍니다.

Input

실제 입력을 받는 부분입니다.

input 태그들의 공통 속성 정리

type
입력 양식 표기
value
양식의 기본 값 설정
name
양식의 이름 표기
autocomplete
자동완성
autofocus
페이지 로드 시 자동 포커스
disabled
입력 불가능 상태 지정
required
필수 입력 항목 지정
placeholder
입력이 없는 상태에서 보여줄 문구

1. text

텍스트 정보를 받는다.

maxlength , minlength 속성을 사용해 최대, 최소 입력 길이를 지정할 수 있습니다.

예시

placeholder를 이용해 입력값이 아직 없는 경우 보여줄 문구를 지정할 수 있습니다.
